# **App Name**: LedgeLog

## Core Features:

- Transaction Logging: Users can quickly enter goods values (e.g., ₹250, ₹50) with an optional note and the entry date.
- Cloud Ledger Sync: Store all transactions securely in a cloud database for access across multiple devices.
- Smart Monthly Rollup: Automatic calculation of monthly expenditures, allowing users to switch between months to see past totals.
- AI Expense Categorizer tool: Use an AI tool to automatically categorize spending entries based on the description notes supplied (e.g., 'dairy', 'hardware', 'office supplies').
- Instant Running Total: Display a dynamic, real-time summation of current entries on the input screen.

## Style Guidelines:

- Primary color: Professional Cobalt Blue (#284DAF) representing trust and financial precision.
- Background: Clean Pearl (#F7F8FA), maintaining a light, accessible, and airy aesthetic.
- Accent color: Vibrant Cerulean (#1C9CCB) used for highlights and positive data indicators.
- Headline font: 'Space Grotesk', providing a tech-forward and structured appearance. Body font: 'Inter', ensuring high legibility for numbers and text.
- Clean, thin-stroke geometric icons to represent currency, dates, and shopping categories.
- A focused single-column dashboard centered on quick input, followed by a prioritized list of recent entries.
- Gentle fade-ins for history updates and subtle pulse effects on the running total change.
